# Ternary Expressions
Ternaries are also [[expressions]] (see [[Statement vs Expression]]), since they are a single [[statement]] that evaluates to a value, one of two, depending on a condition.

```python
result = number / 2 if number % 2 == 0 else (number * 3) + 1
```
