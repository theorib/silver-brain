# 2025-05-21 - Day 45 | Git Branches
## 

Create a new branch and switch to it
```bash
git checkout -b branch-name
```

list all branches
```bash
git branch
```

```bash
git remote add origin https://some-git-repo-address.git
```

Delete branch
```bash
git branch -d branch_name
```

Force delete branch
```bash
git branch -D branch_name
```
