## Sets
### Definition:
A "well-defined" collection of any kind of objects such as people, ideas, numbers, etc.
An **unordered** collection of **unique** objects.

### Mathematical notations.
Sets are enclosed in curly brackets `{` and `}` with values separated by commas `,`, for example:

A set "E" of all even integers smaller than 10: $E=\{ 2, 4, 6, 8 \}$
An empty set is: $Z=\{\}$
or $Z=\varnothing$

Notation for elements in a set: $\in$
Example: $2\in E$
Notation for elements not in a set: $\notin$
Example: $3\notin E$ 
### Cardinality
Cardinality of a set is the number of elements inside a set (or the length of a set) and it is represented with the pipe symbol:
Examples: 
$|E|= 4$ 
$|Z|=\varnothing$ 
$|Z|=\{\}$

### Subset of a set $\subseteq$
$A$ is said to be a subset of $B$ only if every element in $A$ is also an element of $B$. In this case, we write $A\subseteq B$

An empty set is a sub-set of any set.
Any set is a subset of itself, as it satisfies the definition of what a subset is.
