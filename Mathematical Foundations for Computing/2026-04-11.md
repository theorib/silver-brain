## Sets
### Definition:
A "well-defined" collection of any kind of objects such as people, ideas, numbers, etc.
An **unordered** collection of **unique** objects.

### Mathematical notations.
Sets are enclosed in curly brackets `{` and `}` with values separated by commas `,`, for example:

A set "E" of all even integers smaller than 10: $E=\{ 2, 4, 6, 8 \}$
An empty set is: $Z=\{\}$
or $Z=\varnothing$

Notation for elements in a set: $\in$
Example: $2\in E$
Notation for elements not in a set: $\notin$
Example: $3\notin E$ 
### Cardinality
Cardinality of a set is the number of elements inside a set (or the length of a set) and it is represented with the pipe symbol:
Examples: 
$|A|= 4$ 
$|B|=\varnothing$ 
$|B|=\{\}$

### Subset of a set $\subseteq$
$A$ is said to be a subset of $B$ only if every element in $A$ is also an element of $B$. In this case, we write $A\subseteq B$

An empty set is a sub-set of any set.
Any set is a subset of itself, as it satisfies the definition of what a subset is.

### Special Sets $\mathbb N, \mathbb Z, \mathbb Q, \mathbb R$
- $\mathbb{N}$ = Set of **natural numbers** = $\{1,2,3,4\}$
	- A **natural** number is a number that must be a positive whole number, normally excluding $0$
- $\mathbb{Z}$ = Set of **integers** = $\{\dots,-3,-2,-1,0,1,2,3,\dots\}$ 
	- An **integer** is a whole number that can be positive, negative, or zero. It is a number with no fractional or decimal parts.
- $\mathbb{Q}$ = Set of **rational numbers** = (of form $a/b$ where $a$ and $b$ are elements of $Z$ and $b\neq0$) or: 
	- The mathematical notation that represents these is: $$\mathbb{Q} = \{ \frac{a}{b} \mid a, b \in \mathbb{Z}, b \neq 0 \}$$
	- A rational number is any number that can be expressed as a ratio (a fraction) of two integers
- $\mathbb{R}$ = Set of **real numbers**
	- Real numbers represent all numbers of any continuous number line, they can include **rational** numbers as well as **irrational** ($\mathbb{I}$) numbers (these are numbers that cannot be written as simple fractions. Their decimals go on forever, such as $\sqrt 2$ or $\pi \approx 3.14159...$), and certainly all **integers** and **natural** numbers. The only things that are not **real** number are **imaginary** numbers such as $\sqrt -1$
 
 Therefore:  $N\subseteq Z\subseteq Q\subseteq R$

### Additional Special Sets $\mathbb I$ and $\mathbb C$
- $\mathbb I$ = Set of **irrational numbers**
	- Numbers that cannot be written as a simple fraction ba​. Their decimals go on forever, are non-repeating and infinite.
	- _Examples:_ 
		- Famous constants such as: $\pi \approx 3.14159\dots$ or $e\approx 2.71828\dots$ 
		- Non-perfect roots such as $\sqrt{2}$, $\sqrt{3}$ or $\sqrt{5}$
		- Golden ratio: $$\phi = \frac{1 + \sqrt{5}}{2} \approx 1.618$$
- $\mathbb C$ = Set of **complex numbers**
	- Numbers expressed as $a+bi$ where $a$ and $b$ are real numbers and $i$ is the imaginary unit
	- *Examples* $5i, 2+3i, \sqrt −16=4i$
